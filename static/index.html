Go MiniURL
Learning Project  ·  Built with Go & HTMX  ·  2025  ·  Megha Verma

ABOUT PROJECT

A lightweight URL Shortener service that converts a long URL into a clear, mini URL.
As an additional feature, it also allows tracking their click-counts.



TECH STACK

Go   •   Redis   •   SQLite   •   HTMX   •   TailwindCSS



    PROJECT GOALS

  1. Understand internals of this classic system-design problem—covering encoding/decoding, storage, retrieval, and click-tracking.
  2. Apply core system-design concepts in one practical project: caching, rate limiting, pub/sub for analytics, atomic counters, and a Bloom Filter to avoid duplicate URLs—making the project non-trivial and architecture-focused.
  3. Fast development with minimal setup—using Go for quick backend development and HTMX + TailwindCSS for a lightweight JS-free frontend.
