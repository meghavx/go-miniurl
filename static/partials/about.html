A fast, lightweight URL shortener.



CORE FEATURES

  1. Short URL creation & URL redirection
  2. URL data persistence in database
  3. Collision-safe code generation


SYSTEM DESIGN ENHANCEMENTS

  1. Redis caching for fast lookups
  2. Bloom filter to optimize duplicate checks
  3. Layered lookup: Bloom filter → Redis → DB
  4. Token-bucket rate limiting (IP + global)
  5. Redis Pub/Sub for real-time click analytics

TECH STACK

  • Backend: Go
  • Caching & Messaging: Redis
  • Database: SQLite
  • Frontend: HTMX, TailwindCSS


SOURCE CODE

GitHub Link: https://github.com/meghavx/short.ly
