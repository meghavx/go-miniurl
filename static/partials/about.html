A fast and secure URL shortener designed to turn long URLs into clean, shareable links with ease.

ADDITIONAL FEATURES

  • Preview original URL before visiting (safer access)
  • Track usage — total clicks, last visit date and time info

TECH STACK

  • Backend: Go
  • Database: SQLite
  • Caching & Analytics: Redis
  • Frontend: HTML + HTMX , TailwindCSS

SYSTEM DESIGN ENHANCEMENTS

  • PERFORMANCE & SCALABILITY
    • Layered lookup: Bloom filter → Redis cache → DB
    • Bloom filter to optimize duplicate checks
    • Redis cache for fast lookups
    • Real-time analytics via Redis Pub/Sub
  • SECURITY & ABUSE PREVENTION
    • Robust URL validation to block unsafe URLs
    • Rate limiting (IP-level + global)
